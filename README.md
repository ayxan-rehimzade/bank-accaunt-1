# bank-accaunt-1
Bu Java tapşırığında BankAccount sinfi vasitəsilə pul əlavə etmə, çıxarma və balans yoxlama əməliyyatları həyata keçirilir. InputMismatchException, IllegalArgumentException və xüsusi InsufficientFundsException idarə olunur. try-catch ilə səhvlər tutulur, proqram sabit və interaktiv işləyir.
